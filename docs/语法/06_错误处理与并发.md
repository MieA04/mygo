# MyGo 错误处理与并发 (Error Handling & Concurrency)

MyGo 继承了 Go 语言显式错误处理的哲学，但引入了现代化的语法糖来减少重复代码。同时，MyGo 兼容 Go 的并发模型。

## 1. 错误处理 (Error Handling)

MyGo 使用 `error` 接口作为标准错误类型。任何实现了 `Error() string` 方法的类型都可以作为错误。

### 传统方式 (Go 风格)

```mygo
fn readFile(path: string): (string, error) {
    // 模拟错误
    return ("", errors.New("file not found"));
}

fn main() {
    let (content, err) = readFile("test.txt");
    if err != nil {
        fmt.Println("Error:", err);
        return;
    }
    fmt.Println(content);
}
```

### Try-Unwrap 操作符 (`?!`)

MyGo 引入了 `?!` 操作符来简化 `if err != nil` 的模式。
语法：`expr ?! block`。
- 如果 `expr` 返回的最后一个值为 `nil` (无错误)，则表达式求值为除错误外的返回值。
- 如果有错误，则执行 `block`。`block` 必须中断当前控制流（如 `return`, `break`, `panic`）。

```mygo
fn main() {
    // 如果 readFile 成功，content 获取内容
    // 如果失败，执行 {} 内的代码
    let content = readFile("test.txt") ?! {
        fmt.Println("Read failed, using default");
        return; // 必须中断
    };
    
    fmt.Println(content);
}
```

### Panic-Unwrap 操作符 (`?!!`)

`?!!` 用于那些你确信不会失败，或者一旦失败程序就应该崩溃的场景。

```mygo
fn main() {
    // 如果出错，直接触发 panic
    let content = readFile("config.json") ?!!;
    
    fmt.Println("Config loaded:", content);
}
```

## 2. 延迟执行 (Defer)

`defer` 用于确保资源释放（如关闭文件、解锁互斥量）。MyGo 的 `defer` 支持代码块，比 Go 更灵活。

```mygo
fn processFile() {
    let f = openFile("data.txt") ?!!;
    
    // 使用代码块定义复杂的清理逻辑
    defer {
        fmt.Println("Closing file...");
        f.Close();
        fmt.Println("File closed.");
    }
    
    // 处理文件...
}
```

## 3. 并发 (Concurrency)

MyGo 完全兼容 Go 语言的并发模型（Goroutines 和 Channels）。虽然 MyGo 语法层面目前主要关注顺序执行的表达力，但你可以通过调用标准库或 Go 代码来实现并发。

### 使用 Goroutines

(注：MyGo 目前版本可能需要通过互操作调用 Go 的 `go` 关键字，或者使用标准库提供的 `spawn` 函数，具体取决于编译器实现。以下是概念示例)

```mygo
import "time"

fn worker(id: int) {
    fmt.Printf("Worker %d starting\n", id);
    time.Sleep(time.Second);
    fmt.Printf("Worker %d done\n", id);
}

fn main() {
    // 假设有一个 spawn 函数用于启动 goroutine
    // 或者在混合编译中直接调用 Go 定义的启动函数
    for (i : 1..5) {
        // go worker(i); // 如果 MyGo 支持 go 关键字
        // 或者
        // std.spawn(() => worker(i));
    }
}
```

### 通道 (Channels)

可以使用 Go 的 Channel 进行通信。

```mygo
fn main() {
    // 创建 channel
    let ch = make(chan int);
    
    // 发送与接收
    // ch <- 1;
    // let val = <-ch;
}
```

*注意：具体的并发语法细节随 MyGo 编译器版本迭代可能有所调整，建议查看最新的编译器文档或使用 Go 互操作特性。*
