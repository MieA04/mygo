# MyGo 基础语法 (Basic Syntax)

MyGo 是一门旨在平衡表达力与性能的现代编程语言，兼容 Go 语言生态，并引入了诸多现代语言特性（如模式匹配、Trait 系统、空安全操作符等）。

## 1. Hello World

```mygo
package main

import "fmt"

fn main() {
    fmt.Println("Hello, MyGo!");
}
```

## 2. 注释 (Comments)

MyGo 支持标准的 C/Go 风格注释。

### 单行注释
```mygo
// 这是标准的单行注释
```

### 多行注释 (块注释)
```mygo
/* 
   这是标准的
   多行注释 
*/
```

## 3. 变量声明 (Variables)

MyGo 使用 `let` 关键字声明变量，使用 `const` 声明常量。支持类型推导。

```mygo
// 显式类型声明
let age: int = 25;
let name: string = "Alice";

// 类型推导 (推荐)
let score = 98.5; // 推导为 float
let isActive = true; // 推导为 bool

// 可变性：默认所有变量均为可变 (与 Go 一致，但语义上更像 Rust 的 let binding)
age = 26;

// 元组解构声明
let (x, y) = (10, 20);
```

### 常量
```mygo
// 常量为引用不可变，字面量不可变，不限制const声明的结构体内部成员可变性
const PI: float = 3.14159;
const MAX_COUNT = 100;
```

## 4. 基本数据类型 (Basic Types)

MyGo 的基础类型与 Go 语言保持高度兼容：

- **整数**: `int`, `int8`, `int16`, `int32`, `int64`, `uint`, `uint8`...
- **浮点数**: `float` (对应 Go 的 float64), `float32`
- **布尔值**: `bool` (`true`, `false`)
- **字符串**: `string` (UTF-8 编码)
- **空值**: `nil`

```mygo
let a: int = 10;
let b: float = 3.14;
let c: bool = false;
let s: string = "Hello";
```

## 5. 运算符

- **算术**: `+`, `-`, `*`, `/`, `%`
- **比较**: `==`, `!=`, `>`, `<`, `>=`, `<=`
- **逻辑**: `&&`, `||`, `!`
- **位运算**: `&`, `|`, `^`, `<<`, `>>`
- **其他**: 
    - `&`: 取地址
    - `*`: 解引用
    - `++`, `--`: 自增/自减 (作为语句或表达式后缀)
    - `? :`: 三元运算符 (例如 `let max = a > b ? a : b;`)
