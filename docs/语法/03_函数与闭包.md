# MyGo 函数与闭包 (Functions & Closures)

函数是 MyGo 程序执行的基本单元。MyGo 支持具名函数、匿名函数（闭包）以及高阶函数。

## 1. 函数声明 (Function Declaration)

使用 `fn` 关键字定义函数。

```mygo
// 无参数无返回值
fn sayHello() {
    fmt.Println("Hello!");
}

// 带参数和返回值
fn add(a: int, b: int): int {
    return a + b;
}

// 多返回值 (Tuple)
fn divMod(a: int, b: int): (int, int) {
    return (a / b, a % b);
}
```

### 可变参数 (Varargs)
(注：MyGo 语法中暂未显式看到 `...` 语法，但支持切片传递。若兼容 Go，通常支持 `...T`。根据当前语法文件 `paramList` 规则，暂定为切片传递或后续扩展。)

## 2. 函数调用

```mygo
sayHello();
let sum = add(10, 20);
let (div, mod) = divMod(10, 3);
```

## 3. 匿名函数与闭包 (Lambdas / Closures)

MyGo 支持轻量级的 Lambda 表达式语法。

### 语法形式
` (params) : returnType => block `

```mygo
fn main() {
    // 定义一个匿名函数并赋值给变量
    let multiply = (a: int, b: int): int => {
        return a * b;
    };
    
    fmt.Println(multiply(3, 4)); // 输出 12
    
    // 闭包：捕获外部变量
    let factor = 10;
    let scaler = (x: int): int => {
        return x * factor;
    };
    
    fmt.Println(scaler(5)); // 输出 50
}
```

### 作为参数传递
```mygo
fn apply(val: int, op: fn(int): int): int {
    return op(val);
}

fn main() {
    let result = apply(10, (x: int): int => {
        return x * x;
    });
    fmt.Println(result); // 100
}
```

## 4. 泛型函数 (Generic Functions)

函数可以定义类型参数，实现泛型编程。

```mygo
// 泛型函数：交换两个值 (概念示例，MyGo 使用值传递，这里仅展示语法)
fn printIfEqual<T>(a: T, b: T) {
    if a == b {
        fmt.Println("Equal");
    }
}

fn main() {
    printIfEqual<int>(10, 10);
    printIfEqual<string>("hello", "hello");
}
```

可以使用 `where` 子句约束泛型类型（注意：MyGo 中 `where` 子句前置）：

```mygo
where T: Number // 假设 Number 是一个 Trait
fn addGeneric<T>(a: T, b: T): T {
    return a + b;
}
```
